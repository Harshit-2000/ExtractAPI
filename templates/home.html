Documentation

There are two endpoints.
  1. cvExtractAPI-2000.herokuapp.com/api/extract

    • Method allowed - POST
    • Paramaters - files - File/Files in pdf format.
    • Returns - Name, Email, PhoneNo, Experience, Original Text extracted from the CV in json format.
      • Example : 
                      
                    import requests

                    url = 'http://127.0.0.1:5000/api/extract'
                    files = {'files': open('data/sample.pdf', 'rb')}
                    
                    r = requests.post(url, files=files)
                    r.json()

  2. cvExtractAPI-2000.herokuapp.com/api/addname

    • Paramaters - name - first names or list of names to be added seperated by space. Ex - name1 name2 name3
